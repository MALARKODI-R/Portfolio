LASER CUTTING

Introduction

This assignment explores the complete digital fabrication pipeline using Autodesk Fusion 360 and laser cutting technology. The objective was to design a segmented 3D sphere, extract 2D projections in DXF format, and fabricate the model using laser cutting, followed by physical assembly of segments into a spherical form. It demonstrates how complex 3D geometries can be broken down into manufacturable parts using digital tools, and how precise 2D output can be transformed into accurate 3D structures through proper planning.

ROBOTIC ARM

Introduction

This project explores the design and simulation of a robotic arm using Fusion 360 to understand mechanical linkages and motion simulation.

Step-by-Step Guide

1. Base Plate

Design a solid base to support the robotic arm's load.

2. Arm Segments

Create each segment through sketching and extrusion.

3. Joints & Actuators

Use revolute joints to allow rotational motion.

4. Assembly

Assemble parts using As-built Joint commands in Fusion 360.

5. Simulation

Simulate movement by animating joints and checking motion limits.

Challenges

Misalignment

Ensure components align using construction geometry.

Range of Motion

Apply proper constraints to avoid joint over-rotation.

Simulation Errors

Resolve constraints and assembly conflicts before simulation.
