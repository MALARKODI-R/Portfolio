Smart Multi-Functional Bedside Table

The Smart Bedside Table integrates modern technology with essential home furniture, offering a compact, user-friendly, and interactive lifestyle hub. Unlike ordinary bedside tables, it includes embedded electronics and is fully controlled via touch or smartphone app.

Features

  • 📱 Wireless charging pad for smartphones
  • ☕ Peltier-based drink heating/cooling system
  • 💡 Adjustable LED night lamp (touch + app control)
  • 🔊 Built-in Bluetooth speakers
  • ⏰ Smart alarm system (light + sound + vibration)
  • 🔐 Hidden compartments with biometric/RFID access
  • 📱 ESP32 microcontroller + IoT app control
  • 🛞 Lockable wheels for mobility

Problem Addressed

Modern bedrooms often suffer from clutter caused by multiple devices—chargers, alarms, lamps, and more. This table solves that by combining all those features into a single elegant and functional unit, ideal for smart home environments.

Design Description

The structure is compact and ergonomic. Core components are integrated neatly to maintain aesthetics and maximize space:

  • Top surface contains wireless charging and lamp
  • Side houses speakers and compartments
  • Front drawer for essential storage
  • Bottom includes smart power module and lockable wheels

Industrial Applications

  • 🏨 Hospitality – Hotels, Resorts, Smart Rooms
  • 🏥 Healthcare – Patient bedsides with controlled access
  • 🏠 Smart Homes & Automation Products
  • 👵 Elder care & assisted living setups
  • 🎓 Student hostels and compact apartments
