Think Like a Coder

Real-World Problems Solved with Algorithms & Flowcharts

1. The Prison Break

Algorithm:

  • Start
  • Check the dial colour
  • While the dial is not Green:
    • Spin the dial
    • Check the dial colour again
  • Once the dial is Green: Open the door
  • Stop

Flowchart Summary:

  • Start → Check colour
  • While (Colour ≠ Green): Spin the dial
  • If Green: Open the door → Stop

2. Finding the Leader

Algorithm:

  • Start
  • Check if the person has green eyes
    • If no → Reject & check next person
    • If yes → Continue
  • Check if the person has red hair
    • If yes → Check name: "XX"
    • If no → Continue
  • Check if the person wears glasses
    • If yes → Check name has 2 vowels
    • If no → Check name has 3 vowels
  • If all conditions met → Select the person
  • Stop

Flowchart Summary:

  • Start → Check for Green Eyes
  • → Red Hair? → Check Name
  • → Glasses? → 2 Vowels → Select
  • Else → 3 Vowels → Select
  • → Stop

3. The Furnace Bots

Algorithm:

  • Start
  • Take a glance at all shell & furnace numbers
  • While furnace number ≠ 0:
    • Store the furnace number
    • Move to bot whose shell matches that number
  • If furnace number is 0 → It’s the original bot
  • Stop

Flowchart Summary:

  • Start → Read Serial & Furnace Numbers
  • While Furnace ≠ 0 → Move to Bot
  • Furnace = 0 → Found → Stop

4. The Train Heist

Algorithm:

  • Start
  • Initialize position = 0
  • While position ≠ 0:
    • If train moves right (→) → Decrement position
    • If train moves left (←) → Increment position
  • When position = 0 → Press the button
  • Stop

Flowchart Summary:

  • Start → Position = 0
  • While (position ≠ 0): Check arrow direction
  • → ← → Adjust position accordingly
  • Position = 0 → Press Button → Stop
