RFID-Based Attendance System

This system uses an ESP32, RFID reader (RC522), RTC (DS3231), and a servo motor to automate access control. When an RFID tag is scanned, the ESP32 verifies access, logs the entry time and date using the RTC, actuates a servo (e.g., to open a door), and uploads the data to Google Sheets via Wi-Fi for real-time monitoring and logging.

Smart Touch Sensor with Wi-Fi-Based Real-Time Data Logging to Google Sheets

This project uses a capacitive touch sensor to detect touch and control an LED. The sensor's output is monitored using oscilloscopes and displayed in real-time. An ESP8266 Wi-Fi module sends the sensor data to Google Sheets, enabling wireless data logging. This demonstrates basic IoT integration of a touch-based input system with cloud storage.

Real-Time Sensor Data Logging from ESP32 to Google Sheets using MPU6050

This project demonstrates how to log real-time motion data from an MPU6050 accelerometer and gyroscope sensor to Google Sheets using the ESP32 microcontroller and Wi-Fi. The ESP32 reads acceleration and rotation values via I2C, formats the data into JSON, and sends it to a Google Apps Script Web App. The Web App writes the data directly into a Google Sheet with timestamps. This method provides a low-cost, wireless data logging solution ideal for IoT, remote monitoring, and motion analysis.

Servo Trigger Based on Button Count

This Arduino-based project uses a push button to increment a counter shown on a 16x2 I2C LCD. Every 5 presses, a servo rotates to 180° and returns, mimicking an action like gate opening. An LED blinks to indicate the trigger. It demonstrates real-time input handling, display updates, and actuator control. Ideal for automation counters, vending logic, or interactive systems.
